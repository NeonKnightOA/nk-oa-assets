*** spirit3t1 - prejudice *** for quake III arena / cpma by spirit




GENEREAL MAP DATA
==================

MAP NAME     : spirit3t1

LONGNAME     : prejudice

PK3.NAME     : spirit3t1.pk3

GAME TYPE    : supported : TOURNEY, TDM, FFA
               suggested : TOURNEY 1vs1, TDM 2vs2 

MOD          : recommended : CPMA - http://www.promode.org
	       
MAP THEME    : forgotten evil ruins

BASE         : minor modifications to my map 'prejudice' :

                 * replaced YA/bridge with RA
                 * removed RG/bridge
		 * added teleporter from castle front to MH jumppad
		 * shielded RG room from small yard
		 * some ramps for cpm-physics jumps added
		 * 2 of the SGs now appear in TDM only
		 * fixed missing texture bug
		 * fixed texture mis-allignments in same places
		 * many clipping issues fixed
		 * reduced depth of green acid pool
		 * added armor shards near PG
		 * added LG ammo pack in central room
		 * moved some ammo around
		 * added target_location markers for TDM (area names for drawteamoverlay 1)
		 * replaced teleporter texture
		 * much more clipping
		 * replaced red lights by white ones

BUILD TIME   : ~ 2 hours for the adjustments

COMPILE TIME : ~ 3.5 hours @ Pentium M 1500

STUFF USED   : * qeradiant 1.50 RC-1 (linux build)
               * qeradiant 1.4 stable (linux build)
               * q3map2 (linux build)
               * debian gnu/linux
               * cuba libre 

RELEASE DATE : 01-16-2007

FILE SIZE    : 2.2 MB





GAMEPLAY
=========

BOTS         : doom, phobos

WEAPONS      : SG, GL, RL, PG, RG, LG

ITEMS        : 1 YA, 1RA, 1 MH, various health and shards

COMMENTS     : prejudice2 is an open map with lots of small ramps that
               allows fast and furious CPM moves and very high connectivity.
